# Next Word Prediction using LSTM

## Overview

This project focuses on building a next-word prediction system using LSTM (Long Short-Term Memory) for multilingual text including Hindi, Marathi, and English.

## Key Highlights

* Implemented sequence-based prediction using LSTM
* Worked with multilingual datasets
* Applied text preprocessing techniques like tokenization and sequence generation

## Skills Demonstrated

* Python programming
* Machine Learning & Deep Learning
* Natural Language Processing (NLP)
* Sequence modeling
